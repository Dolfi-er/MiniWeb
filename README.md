# MiniWeb (Практика 1)
## 🚀 Быстрый старт

### Предварительные требования

- .NET 8.0, установленный на локальном устройстве
- Fedora 42

### Запуск

#### Клонирование репозитория
- git clone [<repository-url>](https://github.com/Dolfi-er/MiniWeb.git)
- cd MiniWeb

#### Сборка и запуск
dotnet run

### Тестирование

#### Делаем скрипт исполняемым
chmod +x test_api.sh

#### Запуск скрипта
./test_api.sh
